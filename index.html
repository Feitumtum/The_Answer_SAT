The Answer Class
The Answer Class After Class Email Generator
Step 1: Enter Information
A) Insert the relvant information for the School/Teacher Information section
  • For In Person Class --> Fill in the School Name
  • For Virtual Class --> Leave School Name Blank
  • If only one teacher --> Fill in Teacher 1 only

B) Insert the relvant information for the Student/Parent Information
Step 2: Submit/Copy Text
Click the green SUBMIT button to insert changes
  • The correct version of text will be inserted into the email template
Click the green "COPY button to copy to clipboard
  • You can use Ctrl + V to paste into your email
  • Move the Email Subject text after Ctrl + V
Step 3: Repeat for all students
Change the student information and repeat for all student
  • Only fill out the "School/Teacher Information" once per class

  • You only need to change the "Student/Parent Information" and click the Submit button again

Notes

  1. Three sets of pronouns available (she/her, he/his/him, they/their/them)

  2. First line of the output will be the subject line for the email.

  3. Clicking the COPY TEXT button will copy all text (including the email subject and logo).
