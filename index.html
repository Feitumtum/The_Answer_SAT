The Answer Class After Class Auto Emails

How to use

  • This is an tool to help teachers of The Answer Class to fill out the after class emails to parents.

  • All you have to do it insert the relvant information for each student and the correct version of text will appear so that you can copy and paste into your email draft (picture will also copy over) Once you are done with one student, you can simply change the student information and the school and teacher information will remain the same.

  • There are three sets of pronouns available to use based on the student's preferred pronous

    • he/his/him
    she/her
    they/their/them

  • For virtual vs in person classes, there will be difference sentences.

    If virtual, you only need to fill out the date Ex) AT prep class that Bella attended on [DATE OF CLASS]
    If in person, you only need to fill out the school Ex) SAT prep class that Bella attended at [SCHOOL]

  • The first line of the output will be the subject line for the email.
